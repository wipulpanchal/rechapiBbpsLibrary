# rechapiBbpsLibrary
Rech api Bills Library

>Step 1. Add the JitPack repository to your build file
```gradle
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
```gradle
>Step 2. Add the dependency	in gradle
dependencies {
	        implementation 'com.github.wipulpanchal:rechapiBbpsLibrary:1.0.0'
	}
  dependencies {
	        implementation 'com.github.wipulpanchal:rechapiBbpsLibrary:1.0.0'
	}
  ```
